Conceitos de Web Design

HTML
Linguagem de marcação usada para estruturar páginas da web. Define títulos, parágrafos, imagens e outros elementos.
CSS
Linguagem usada para estilizar páginas HTML, controlando cores, tamanhos, layouts e aparência visual.
Tag
Elemento usado no HTML para marcar partes do conteúdo, como <p>, <h1> e <img>.
Elemento
É a estrutura completa no HTML, incluindo tag de abertura, conteúdo e tag de fechamento.
Web Designer
Profissional responsável por criar o visual e a experiência do usuário em sites, combinando design e tecnologia.
